# Публичные API на публичных данных
Идея в том чтобы наебашить публичные API, которые достают инфу из распределенных БД высокомасштабируемых с слабой согласованностью <br>
Предполагается что данные будут обновляться не часта <br>
Это GeoIP, данные государств, банков, любые данные например из Excel таблиц <br>
Данные в БД редко обновляются <br>
Выручка за счёт донатов? <br>
Rate limit ебашить можно и другие признаки <br>
Простая в реализации идея <br>
